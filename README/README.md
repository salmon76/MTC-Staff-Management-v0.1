# LINE LIFF Web App – PoC Project

This project is a Proof of Concept (PoC) for building a Fullstack Web Application integrated with LINE LIFF and LINE Messaging API using Next.js and Supabase.

The goal is to validate end-to-end integration:
LINE App (LIFF) → Web Frontend → Backend API → Database → LINE Bot

---

## 1. Tech Stack

- Frontend: Next.js (App Router)
- UI: Tailwind CSS
- Backend: Next.js API Routes (Node.js runtime)
- Database: Supabase (PostgreSQL)
- LINE Integration:
  - @line/bot-sdk (Messaging API)
  - @line/liff (LIFF SDK)
- Deployment: Vercel
- CI/CD: GitHub Integration

---

## 2. Prerequisites

Before starting, make sure you have:

- Node.js >= 18
- npm or yarn
- Git
- LINE Developer account
- Supabase account
